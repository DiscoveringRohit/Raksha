# Raksha

Raksha Android Application.

## Future Prospects

- **Fake Trigger via Pitch Control**: Implement a mechanism to activate a fake SOS trigger using voice pitch control.
- **Panic Detection & Cloud Recording**: Utilize panic detection algorithms to automatically capture a 10-second voice recording during emergencies, securely stored via Firebase.
- **Enhanced Microphone Sensitivity**: Apply audio gain control to increase the range for sound detection, overcoming the physical limitations of standard smartphone microphones that struggle to pick up audio from a distance.
